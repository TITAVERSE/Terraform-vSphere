# Terraform-vSphere

This module is an unversal module to create and manipulate one and only one resource in VMWare vSphere environment.

# Prerequisites
- Terraform version : 1.7.2
- vCenter version : 7.X o r8.X
- Terraform vSphere version : 2.3.1
